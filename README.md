Análise da Pandemia de COVID-19 no Brasil com base em dados públicos.

                                                  Objetivo 


Investigar a evolução da pandemia, a distribuição dos casos e o impacto da vacinação utilizando análise de dados e visualizações interativas.

                    Fontes de Dados

- [Our World in Data – COVID-19](https://ourworldindata.org/covid-deaths)
- Arquivo CSV: [owid-covid-data.csv](https://covid.ourworldindata.org/data/owid-covid-data.csv)

      Ferramentas Utilizadas

- Google Colab (análise e tratamento)
- Looker Studio (visualizações)
- Google Sheets (integração com painel)
- GitHub (documentação e entrega)

     Painel Looker Studio
 Apresenta:

- Scorecards com totais de casos e mortes
- Evolução diária e mensal de mortes e vacinação
- Distribuição dos casos por período (Pré e Pós-vacinação)
- Tabela dinâmica mensal

                                                    Conclusão 
A análise mostra que o avanço da vacinação está diretamente ligado à redução nas taxas de mortalidade. Visualizar esses dados facilita a compreensão e tomada de decisões baseadas em evidência.
